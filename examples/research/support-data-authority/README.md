# Trace-grounded support loop demo

This example is consumed by the Core Tilt integration. Core seeds the synthetic
`<entity>/northstar-support-agent` Weave project, while Fugue allowlists that
exact project and maps selected call provenance to the reviewed
`support-data-authority-v1` recipe.

The trace selection never becomes executable task content. The experiment uses
the checked-in synthetic paired support task and binds the selection, trace-root
snapshot, recipe, verifier, three loop designs, runtimes, and six-cell plan into
one operator-approved preview digest.
